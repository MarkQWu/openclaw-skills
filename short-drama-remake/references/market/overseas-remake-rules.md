# Overseas Remake Rules

Purpose: define the target-market hard boundaries for overseas remake projects.

This file is a remake-native scaffold. It does not import original `short-drama` overseas runtime files; it re-expresses only the adaptation categories needed to keep `/仿写 出海` from producing translated domestic concepts.

## Required Separation

- Preserve story function, not domestic surface mechanics.
- Do not import or directly read `short-drama/references/overseas/*` inside remake nodes.
- Use this file only through the remake target-market adaptation layer.
- Classify every new market rule through `layer-taxonomy.md` before it can become a blocker.

## Active Remake Boundaries

1. **Target market must be explicit**: "overseas" alone is not enough. Name the platform, region, language audience, or working assumption before adapting the concept.
2. **No source-market mechanism carryover**: domestic status systems, family/legal/workplace/school mechanics, medical misunderstandings, and humiliation-to-power arcs must be converted into target-market power grammar before planning.
3. **No original-skill hard-rule import**: if an original `short-drama` overseas rule seems useful, re-express it here as R1-R5 adaptation logic with source function and target replacement named.
4. **No protected expression retention**: names, title logic, signature reveal order, line turns, visual motifs, and episode sequence from the source must be replaced, not translated.
5. **Relationship grammar must be declared**: adaptation reports must name the target-market agency / consent / coercion boundary before any dark-romance, mafia, cartel, forced-proximity, or revenge premise proceeds.
6. **Filmability is replacement logic, not style polish**: when a source beat relies on crowd blocking, wide spectacle, or unfilmable action, map the viewer function into a target-market vertical equivalent.
