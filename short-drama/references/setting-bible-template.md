## verified 字段定义（修订 2）

bible 中所有事实字段都带 `verified: true` 或 `[待人工核源]` 标记。准入规则见 `references/research-guide.md#bible-字段-verified-准入门槛`。

剧本写作时：
- `verified: true` 字段 → /分集 可直接引用
- `[待人工核源]` 字段 → /分集 引用时必须加 `[考据待定]` 标注，/自检 第 8 维度本条不扣分但提示用户
- `[来源冲突]` 字段 → /分集 不允许引用，必须先 /考据 修复冲突

## token 预算

bible 全文目标控制在 **5000 token 内**（约 3000-3500 中文字）。实测样本 ~2835 token / 5000，剩余 43%（详见 H2 验证报告）。

超出策略（修订 7）：
- 简单场景：bible 不分层
- 极复杂混合题材（如"现代医生穿越古代")：拆为「核心 bible」（术语 + 雷区，~3000 token，每集必加载）+「扩展 bible」（常识锚点 + 物资清单，按需加载）
- 默认不启用分层
