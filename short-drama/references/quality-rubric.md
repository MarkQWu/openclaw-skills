# 质量自检评分标准（Rubric）

> 来源：红果编剧第一课 10 讲提炼 + 行业实践。`/自检` 命令加载本文件。
> All Rights Reserved — 本文件为方法论知识层，未经许可不得复制分发。

## 评分方法论

1. **先推理再评分**：每个维度先列出从剧本中观察到的具体事实，再对照锚点给分
2. **整数分制**：不允许小数
3. **锚点校准**：参照各维度的分数锚点描述，找到最接近的锚点
4. **硬约束优先**：触发硬约束时分数受上限限制，无论其他表现多好

## 分值分配（总分 75 分）

| 维度 | 分值 | 类型 | 说明 |
|------|------|------|------|
| 节奏 | 1-10 | 核心 | — |
| 爽点 | 1-15 | 核心（加权） | 短剧商业成功的核心驱动力，权重最高 |
| 台词 | 1-10 | 核心 | — |
| 格式与可拍性 | 1-5 | 门槛 | 达标即可，不是区分因素 |
| 主线与连贯性 | 1-10 | 核心 | — |
| 反抽象与镜头化 | 1-10 | 核心 | — |
| AI Slop | 1-10 | 核心 | — |

**为什么爽点加权**：格式规范是必要条件但不区分好坏——格式差不过稿，格式好不代表剧本好。爽点密度和质量才是短剧留存率和商业表现的核心驱动力。

## 一、节奏

### 检查项

#### 开场检查（第03讲）

- [ ] 人设建立：身份/地位/情绪通过具体事件展示，非旁白说明
- [ ] 主线目标：目标在前两集直白展现，不藏不绕
- [ ] 困境代入：即时、具体、有压迫感的危机
- [ ] 情绪推进：密集戏剧性情绪转换（非平铺直叙）
- [ ] 信息密度：台词承载多重功能（推剧情+揭关系+暴矛盾+强人设）

#### 开场禁忌（第03讲）

- [ ] 无旁白/内心独白铺背景
- [ ] 无冗余对话浪费开场时间
- [ ] 无平庸温吞开场（照搬原著式推进）
- [ ] 人设三问清晰：我是谁？要什么？面临什么危机？

#### ⚡ 三锚点检查（硬约束）

- [ ] 0-3s 冲突点：前 3 段内出现冲突/钩子
- [ ] 30s 情绪爆破：第 2-3 场次存在情绪高点
- [ ] 结尾钩子：有悬念标记（张力台词或冲击力画面）
- **缺少任意一个锚点 → 该维度 ≤5 分**

**题材适配**：甜宠/日常向的"0-3s 冲突点"可替换为"0-3s 情感钩子"（如心动瞬间、误会引发、反差萌）。核心是前 3 段内抓住观众，不限于冲突。

#### 单集节奏（第05讲）

- [ ] 前 30 秒：背景+困境+动机三合一（「前30秒定生死」）
- [ ] 中间 1 分钟：主线推动 + 人物关系 + 新伏笔
- [ ] 最后 30 秒：强悬念/未完成动作/反转
- [ ] 无"信息真空"过渡场景
- [ ] 无无效嘴炮拖沓段落
- [ ] 每集有实质性进展（非重复停滞）

#### 结尾钩子（第06讲）

- [ ] 钩子类型明确：张力台词（未说完的质问/身份暗示/情绪化宣言）或冲击力画面（对视未开口/手伸未触/门推一半）
- [ ] 遵循"高潮切开，一半留下集"原则
- [ ] 无强行加戏或内心独白复述目标
- [ ] 钩子是情节到高潮临界的自然停顿

#### 全剧大节奏（仅 /自检 all 时检查，第05讲）

- [ ] 1-10 集黄金窗口：主线确立 + 核心看点亮出 + 观看动机建立
- [ ] 11-30 集：主线实质推进，无重复停滞，适度引入支线
- [ ] 31-80 集：伏笔揭晓，多轮反转，情绪集中释放
- [ ] 题材节奏匹配：复仇/逆袭→快狠准；甜宠→误会-吃醋-和解-升温

### 分数锚点

| 分数 | 描述 |
|------|------|
| 2 | 开场超过 5 段才出冲突，中段有信息真空，结尾无钩子；三锚点缺失 2+ 个；用旁白铺背景 |
| 5 | 开场有冲突但五要素缺 2+，或前 30 秒未完成三合一；中段有拖沓嘴炮；结尾钩子力度弱（内心独白复述） |
| 8 | 五要素齐全，前 30 秒高效完成三合一；节奏紧凑无一处拖沓；三锚点全部达标；钩子有效 |
| 10 | 开场前 3 段即强冲突+多重反转，中段层层递进如多米诺骨牌，结尾钩子让人「划不走」；节奏如过山车 |

### 避坑（低分特征，第03/05讲）

- 节奏拖沓：短剧一集仅 500-800 字，不在"展开"而在"爆破"
- 受众错位：写情绪前先问「写给谁看」
- 情绪空洞：机械套用下跪/扇巴掌三件套（观众共情的是「尊严被践踏」不是具体行为）
- 桥段窠臼：不反复使用同类型桥段（桥段不怕老，怕的是表达旧）

## 二、爽点（满分 15 分）

### 检查项

#### 爽点数量与强度（第07讲）

- [ ] **硬约束：≥3 个爽点**，不足 3 个 → 该维度 ≤6 分
- [ ] 爽点类型多样（不全是同一类型）
- [ ] 爽点强度达标（有明确的情绪释放，非轻描淡写）

#### 题材分化（硬约束适配）

爽点的定义随题材变化，评分时需按题材标准：
- **爽感型**（战神/逆袭/复仇）：身份碾压、打脸、逆袭翻盘为核心爽点
- **情感型**（甜宠/日常/治愈）：心动瞬间、误会冰释、双向奔赴、守护行为也算爽点
- **悬疑型**（推理/反转）：真相揭露、反转震撼、逻辑闭环也算爽点

不能用爽感型标准评判甜宠剧本的爽点数量。

#### 四大爆点类型（第07讲）

- [ ] 逆袭成功：小人物崛起有合理理由（奇遇/金手指），非凭空
- [ ] 反转打脸：「意料之外，情理之中」——前期有伏笔铺垫
- [ ] 意外收获：有因果铺垫，无悬浮感（凭空获得）
- [ ] CP 甜宠：男频侧重保护宠爱，女频强调双向奔赴

#### 期待管理（第06讲）

- [ ] 大期待（终极目标）：最晚第二集直白展现，具体化（「三个月赚一千万」优于「成为有钱人」）
- [ ] 小期待（阶段性任务）：大目标拆解为关卡系统，达成时观众感知进步
- [ ] 关键节点反复强化期待

#### 悬念设计（第06讲）

- [ ] 使用至少 2 种信息差类型组合
- [ ] 悬念无逻辑硬伤，与剧情紧密结合
- [ ] 无强行反转

#### 痛点铺垫（第07讲）

- [ ] 痛点具象化：「尊严被践踏」→「酒席上被泼酒」
- [ ] 男频痛点对齐：安全感/尊严/保护家人
- [ ] 女频痛点对齐：情感安全感/自我价值

#### 困境情节三原则（第07讲）

- [ ] 时长简洁：不长期停留低谷
- [ ] 留希望：暗示反击准备
- [ ] 针对性强：来自清晰对立角色

### 分数锚点

| 分数 | 描述 |
|------|------|
| 3 | 爽点 <3 个或全无；痛点未铺垫直接爽；期待模糊（观众不知主角要什么）；反转凭空无伏笔 |
| 6 | 有 3 个爽点但类型单一（全是打脸）；大期待有但不具体；痛点有但未具象化；困境停留过长 |
| 9 | 3+ 爽点且至少 2 种类型；大期待具体清晰+小期待关卡化；痛点具象+爽点释放形成完整弧线 |
| 12 | 爽点类型丰富且层层递进；期待管理精准（观众始终知道「距离胜利还有多远」）；痛点-困境-爆发节奏完美 |
| 15 | 在 12 分基础上：每个爽点都有前期铺垫+释放+观众情绪共振；蓄力-释放节奏让人「看了还想看」 |

### 避坑（低分特征，第07讲）

- 情绪自嗨：违背受众核心诉求（男频写男主为爱牺牲被视为愚蠢）
- 逻辑缺失：强行推进，人物行为不合逻辑
- 打脸对象错误：应针对主反派，不可欺凌路人/践踏女性/侮辱公职人员

## 三、台词

### 检查项

#### 功能密度（第03/09讲）

- [ ] 台词承载多重功能：推进剧情+揭示关系+制造冲突+强化人设
- [ ] 推动剧情是第一要务（揭示新信息/制造冲突/做关键决定）
- [ ] 台词是人物性格的外化（同一动作因性格不同说法不同）

#### 精简与口语化（第09讲）

- [ ] 「刀刀见血」无冗余废话
- [ ] **台词长度：逐条扫描，标出超过 2 句的台词并建议拆分**
- [ ] 口语化自然（非书面语）
- [ ] 对白为主力 → 画外音辅助 → 内心独白最少

#### 角色区分度（第09讲）

- [ ] 每个角色有辨识度（换角色说也行 = 失败）
- [ ] 称呼匹配层级关系（上级→小X/直呼名；下级→X总/X老师）
- [ ] 角色-语言风格映射表一致

#### 题材语感（第09讲）

- [ ] 古装/奇幻：用词稍文雅但需易懂
- [ ] 情感题材：重细腻度和情绪感染力
- [ ] 轻喜剧：笑点源于性格和情境，避免堆砌过时梗

#### 质感提升（第09讲）

- [ ] 方言/口头禅/行业术语适度点缀
- [ ] 网络热梗谨慎使用（适度拉近但不滥用）

### 分数锚点

| 分数 | 描述 |
|------|------|
| 2 | 台词拗口冗长，心理描写当台词念；角色说话无区分度；大量书面表达；霸总说「我认为我们需要重新审视这段关系」 |
| 5 | 台词通顺但功能单一（只推剧情不揭关系）；角色有区分但不鲜明；偶有超 2 句长台词 |
| 8 | 台词多功能、精简有力；角色有 DNA（能听出谁在说）；称呼/语感匹配题材；无超长台词 |
| 10 | 台词即人设，金句能被观众截图传播；「删掉 1/3 核心信息还在」；每句都承载多重功能如「刀刀见血」 |

### 自检方法（第09讲）

- 大声朗读检验自然度
- 试着删除 1/3 看核心信息是否还在

### 三大误区（第09讲）

1. 台词拗口篇幅过长 → 简化信息量，用符合身份的口吻
2. 心理描写过多（OS 当台词念）→ 用动作+简短台词替代
3. 台词空洞脱离逻辑 → 通过具体细节让人物逐步理解处境

## 四、格式与可拍性（满分 5 分，门槛型）

> 门槛型维度：达标即可（3-5 分），不达标直接拉低总分（1-2 分）。格式好不代表剧本好，但格式差一定过不了稿。

### 检查项

#### 格式规范（第01讲）

- [ ] 场景头完整：集数标注、时间（夜/日）、场景位置（外/内）、场景编号
- [ ] 人物动作用 △ 标记
- [ ] 语气/停顿有标注
- [ ] 景别标注存在
- [ ] 音乐/特殊效果有提示

#### 可拍性评估（第08讲）

- [ ] 场景数量合理：集中拍摄可行，通过布景微调区分
- [ ] 核心人物 3-5 人，总角色不超 20 人
- [ ] 高潮集新角色 ≤2 个
- [ ] 日常集避免引入新角色
- [ ] 无过度依赖高难度元素（大量动作戏/特效/群演）
- [ ] 高危镜头（水下/高空/爆破）仅在核心卖点时使用
- [ ] 单集字数在 500-800 字范围

### 分数锚点

| 分数 | 描述 |
|------|------|
| 1 | 场景头大量缺失；角色超 20 人；无法执行拍摄 |
| 2 | 场景头不规范；无景别标注；字数严重超标；有不必要的高危镜头 |
| 3 | 格式基本规范但有遗漏（缺音乐提示/景别）；角色略多但可控 |
| 4 | 格式完整规范；核心 3-5 人+总角色合理；场景可集中拍摄 |
| 5 | 格式专业精确如工业标准；场景/角色经济高效；「拿到剧本就能直接开拍」 |

### 关键论述（第08讲）

> 「一个能顺利拍出来的 80 分剧本，价值远大于无法落地的 100 分梦想。」

## 五、主线与连贯性

### 检查项

#### 主线清晰度（第04讲）

- [ ] 灵魂三问通过：一句话讲清（主角+目标+阻碍+亮点）？主角一秒共情？故事为谁而写？
- [ ] 目标具体有情绪张力（「首富男主重生归来向背叛兄弟复仇」而非「男孩重生复仇的故事」）
- [ ] 阶段性目标层层递进（关卡系统）
- [ ] 支线为主线"助攻"，不喧宾夺主
- [ ] 大纲聚焦改变命运的关键事件，非时间流水账

#### 前后一致性

- [ ] 与前后集无矛盾
- [ ] 角色行为一致（无突然降智/性格突变）
- [ ] 伏笔有延续和回收

#### 转变逻辑（realism-checklist.md）

- [ ] 认知转变有具体触发物（看到短信/听到对话/撞见场景，非凭空感觉）
- [ ] 情绪转折有过渡状态（信任→[犹豫/困惑]→翻脸，非直接跳变）
- [ ] 关系呈现生活化（「两家月聚一次餐」而非「多年好友」）

### 分数锚点

| 分数 | 描述 |
|------|------|
| 2 | 主线模糊（无法一句话说清）；前后矛盾明显；角色行为跳跃无逻辑；支线喧宾夺主 |
| 5 | 主线有但不够具体（「想复仇」但未说清如何/对谁）；转变缺中间状态；有 1-2 处小矛盾 |
| 8 | 主线一句话清晰+目标具体；转变有触发物和过渡；支线服务主线；前后一致 |
| 10 | 主线如「脊柱」支撑全集；阶段性目标如关卡层层递进；每个转变都有具体触发+合理过渡；伏笔精准回收 |

### 常见误区（第04讲）

1. 大纲写成流水账 → 应只聚焦改变命运的关键事件
2. 副线喧宾夺主 → 支线为主线助攻
3. 幸存者偏差 → 「新人唯一能把握的就是清晰有力的主线」

## 六、反抽象与镜头化

### 检查项

#### 反抽象（硬约束）

- [ ] **△ 段落无纯情绪词**：扫描全文，逐个标出「她很伤心」「气氛凝重」「他很愤怒」类描述
- [ ] 每处纯情绪词给出物理化替换建议（情绪→具体动作/表情/行为）

**纯情绪词判定边界**：
- ❌ 纯情绪词："她很伤心"、"他很愤怒"、"气氛凝重"、"他很紧张"（无物理表现）
- ✅ 不算纯情绪词："气得脸色铁青"（有物理表现"脸色铁青"）、"杀意凛然地拔刀"（有动作"拔刀"）
- **判断标准**：同一描述中是否包含可被摄影机拍到的物理表现。有 → 不算；无 → 算

#### 镜头化思维（第08/10讲）

- [ ] 少写心理多写行为——通过表情/动作/沉默传达情绪
- [ ] 关键信息通过剧情/对白/行为呈现，不依赖观众脑补（无上帝视角）
- [ ] 大段静态描写已转化为外化动作与精炼对话
- [ ] 「简单直给」：开场十几秒内让观众理解看点

### 分数锚点

| 分数 | 描述 |
|------|------|
| 2 | 大量纯情绪词（≥5 处）；依赖心理描写推进故事；关键信息靠观众脑补；有大段静态描写 |
| 5 | 有 2-3 处纯情绪词；部分场景用心理描写代替行为；偶有上帝视角 |
| 8 | 全文无纯情绪词（或仅 1 处已标出）；情绪通过物理动作传达；信息通过剧情/对白呈现 |
| 10 | 每个情绪点都有画面感的物理表达；「读每句话脑中能浮现具体镜头」；零心理描写零上帝视角 |

### 转化参考（第01/08/10讲）

| 抽象表达 | 镜头化表达 |
|---------|-----------|
| 她很伤心 | 她咬住下唇，指甲掐进掌心 |
| 气氛凝重 | 所有人停下筷子，只有墙上的钟在响 |
| 他很愤怒 | 他把杯子摔在地上，碎片溅到对方鞋面 |
| 她犹豫了 | 她的手伸到门把手上，停了三秒，又缩回来 |

## 七、AI Slop

### 检查项

#### 台词书面化

- [ ] 扫描全部台词，标出书面表达：「我认为我们应该」「在某种程度上」「不可否认」「事实上」「从某个角度来说」
- [ ] 每处给出口语化替换

#### 情绪转变过平滑

- [ ] 检查角色情绪/立场转变是否缺少抗拒、犹豫、反复等中间状态
- [ ] 「她听完立刻原谅了他」= AI Slop，需要加入挣扎过程

#### 巧合堆砌

- [ ] 统计「正好」「恰好」「刚好」「碰巧」「没想到」出现次数
- [ ] **全集 ≥3 次 → 判定巧合过多**，每处标出并建议用因果逻辑替代

#### AI 生成痕迹（补充）

- [ ] 无过度完美的角色（全面优秀无缺点）
- [ ] 无过度对称的情节结构（机械化的三段式）
- [ ] 无「总结性」结尾台词（角色像在做 presentation）

### 分数锚点

| 分数 | 描述 |
|------|------|
| 2 | 书面表达 ≥5 处；情绪转变全部一步到位无中间态；巧合词 ≥5 次；明显 AI 生成质感 |
| 5 | 书面表达 2-3 处；有 1-2 处情绪跳变；巧合词 2-3 次；偶有 AI 痕迹 |
| 8 | 书面表达 ≤1 处；情绪转变有过渡；巧合词 ≤1 次；读起来像人写的 |
| 10 | 零书面化、零巧合堆砌；情绪转变层次丰富（抗拒→犹豫→松动→接受）；完全无 AI 痕迹 |

## 红果过稿预估

基于总分（75 分满分）：

| 总分区间 | 预估 | 说明 |
|----------|------|------|
| 60-75 | ✅ 大概率过稿 | 各维度均衡，无硬伤 |
| 45-59 | ⚠️ 需修改重点项后可过 | 修复【严重】问题后有机会 |
| 30-44 | ❌ 过稿困难 | 多维度需大改 |
| <30 | 🚫 建议重写 | 导出阻断 |

> 注意：过稿预估基于红果编剧第一课提炼的方法论标准，非平台官方审核算法。实际过稿还受题材热度、平台当期需求、制作方资质等因素影响。
