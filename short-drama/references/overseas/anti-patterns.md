# Overseas Anti-Patterns Navigation

Purpose: route overseas-mode failure patterns to the correct reference layer. This file is a navigation surface, not the body repository for every rule.

## Read Order

1. **Domestic carryover and failed transfer** -> `anti-domestic-transfer.md`
   - Use when diagnosing Chinese short-drama mechanisms, male-frequency satisfaction arcs, domestic self-check rubrics, or platform assumptions leaking into overseas mode.
2. **Structure vocabulary pollution** -> `anti-structure-import.md`
   - Use when reviewing outlines, beat sheets, or reference docs for Hollywood feature-film structure imports such as three-act language or Save the Cat terms.
3. **9:16 filmability and vertical craft** -> `vertical-filmability.md`
   - Use when reviewing scenes for phone-screen readability, shot design, action blocking, VO misuse, and static dialogue.
4. **Legal, IP, similarity, likeness, and cultural risk** -> `compliance-risk.md`
   - Use when a beat, character, title, visual motif, source resemblance, real-world institution, protected class, or AI likeness issue could block delivery.

## Layer Boundary

- L3/L4 craft failures belong in `anti-domestic-transfer.md` or `anti-structure-import.md` only when they describe audience grammar, story pressure, or structural pollution.
- L5 medium failures belong in `vertical-filmability.md`.
- L7 delivery risk belongs in `compliance-risk.md`, even if the risky material first appears during craft review.
- L8 anti-pollution rules belong here only as routing notes; body rules live in their layer files.

## Quick Gate

If a rule says "this cannot legally or safely ship," route it to `compliance-risk.md`.

If a rule says "this is imported from the wrong market or wrong story system," route it to `anti-domestic-transfer.md` or `anti-structure-import.md`.

If a rule says "this cannot be shot or read in vertical video," route it to `vertical-filmability.md`.
