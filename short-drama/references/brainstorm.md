# 同构创作：换皮四步骨架模板

> 本文件为 `/重构` 命令的方法论参考。
> 换皮不是抄袭——是提取结构逻辑、替换题材外壳，用相同骨架讲完全不同的故事。

---

## 换皮四步骨架定义

```
第1步 开局钩：[隐形身份/信息差切口/特殊能力藏匿形式]
第2步 捶杀机：[核心压迫机制——什么力量在打压主角？以什么方式？]
第3步 倒戈点：[什么触发中立/敌方转向主角？铁证/公开事件/代价揭露]
第4步 绝杀式：[终极审判的执行方式——谁执行？用什么武器？以谁的名义？]
```

### 各步骤详解

| 步骤 | 核心问题 | 常见形式 |
|------|---------|---------|
| 开局钩 | 主角隐藏了什么？观众知道但反派不知道 | 隐藏身份（退役战神/隐世神医）/ 信息差（重生者知道未来）/ 能力封印（被压制的天赋） |
| 捶杀机 | 谁在打压主角？用什么手段？打压持续多久？ | 家族排斥 / 职场欺凌 / 豪门歧视 / 婆家碾压 / 系统性偏见 |
| 倒戈点 | 什么事件让敌方/中立方站到主角一边？ | 铁证被掌握 / 主角身份公开揭露 / 靠山发现被欺骗 / 公开羞辱事件 |
| 绝杀式 | 谁执行终局清算？清算手段是什么？以什么权威名义？ | 更高权力介入（皇权/军方/集团总裁）/ 被害人直接反制 / 系统性一锅端 |

---

## 同构变体生成规则

**保持不变的：**
- 四步骨架的逻辑顺序（开局钩→捶杀机→倒戈点→绝杀式）
- 信息差结构（观众知道 > 主角知道 > 反派知道）
- 情绪节奏（隐忍蓄力→中段捶杀→靠山倒戈→终局清算）

**可以替换的：**
- 题材赛道（都市→古装宫廷 / 战神→神医 / 重生→末日 / 豪门→宫斗）
- 压迫机制（家族→权贵集团；丈夫→婆家；上司→资本；皇帝→权臣）
- 身份外壳（退役战神→落魄神医 / 重生者→穿越者 / 豪门继承人→皇室私生子）
- 倒戈触发方式（同类型可互换：铁证公开 / 揭露欺骗 / 阵营利益重新算）

## 骨架提取示例

**参考：** 《战神归来》（都市男频）

```
开局钩：退役特种兵隐藏身份，被当废物赘婿对待
捶杀机：岳父家族系统性羞辱，反派借家族资源步步碾压
倒戈点：军方长官公开出现 + 反派搞到铁证被主角掌握
绝杀式：军方名义清算 + 家族内靠山切割反派 + 公开处刑
```

**同构变体（古装宫廷）：**

```
开局钩：落魄皇子隐藏太子身份，在后宫被各方势力踩踏
捶杀机：权臣派系系统性排挤，后宫嫔妃联手欺压
倒戈点：皇帝密诏公开 + 权臣内部有人拿到把柄倒向太子
绝杀式：皇帝亲自赐死权臣 + 太子身份公告天下
```

## 使用提示

- `/重构` 只生成创意骨架，不建项目文件
- 确认选用某变体后：`/新建` 建项目 → `/策划` 展开完整剧本
- 骨架越简洁，变体越容易生成——4 句话足够，不要过度细化
- 同一骨架可以生成 3-5 个变体，再挑市场潜力最大的一个深入
