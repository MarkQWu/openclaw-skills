# 短剧 skill anchor 灰度库（Anchor Library）

> **本文件用途**：为 short-drama skill 的 implicit knowledge 触发机制提供 33 部中文现象级 IP 的 anchor 数据。
>
> **当前激活状态（v1.15.0）**：
> - **Phase 1 MVP 激活**：5 部 anchor（《何以笙箫默》《庆余年》《后宫·甄嬛传》《平凡的世界》《隐秘的角落》），已写入 `genre-guide.md` 对应 5 个题材末尾的 `### anchor 参考` section
> - **Phase 2 灰度库**：本文件其余 28 部 anchor 是灰度激活源，**Phase 1 不参与触发**。MVP A/B 测试通过后，按数据驱动批量激活到 `genre-guide.md`
>
> **触发哲学**：所有 anchor 借给短剧模型的是**世界观厚度 / 角色调性 / 题材想象力 / 情绪锚点**，**不借节奏**——节奏由 short-drama skill 自身硬约束（秒级节奏锚点 / 台词≤2 句 / 爽点≥3）控制。
>
> **数据源**：本库内容来自 4 轮调研合并产出（v2 final-merged-v2.md），已经过独立审查修正：
> - 《延禧攻略》编剧统一为「周末（主笔编剧）+ 于正（编审 / 总制片人）」
> - 《白夜追凶》归属修正为「指纹（韩冰）」，从紫金陈宇宙剔除
> - 《道诡异仙》评级 D → B（中国作家网 + 阅文 IP 盛典 2025）
> - 《遮天》《师兄啊师兄》评级修正
>
> 跨题材重复条目处理：唯一归类（庆余年→重生穿越，延禧攻略/甄嬛传→古装宫廷），但保留跨题材"借不同 hint"的交叉提示
