# Comic 漫剧规则（medium=comic）

本文件适用于 `medium=comic`——动态漫画 / AI 漫剧 / 原生漫剧统称。仿真人 AI 剧规则见 `ai-live-rules.md`，跨介质通用规则仍在 `quality-rules.md`。

## 加载时机

执行前读取 `.drama-state.json#medium`：
- `"comic"` → 加载本文件
- `"ai_live"` 或字段缺失 → 加载 `ai-live-rules.md`

## 设计原则

漫剧的情绪承载层是：**分镜特写 + 对白 + OS/VO 情绪标签**。因为 AI 漫剧人物面部微表情表达弱于真人，对白承担更多情绪表达职责；分镜切片可承载长台词和单场景内的情绪递进。场景数 ≤3 的资产集中度约束换来分镜密度空间。

AI 漫剧 vs 原生漫剧 ≈ 2025 年后语义融合度 >80%，本文件不做分化——共用规则。

## 硬规则（违反 = 强制返工）

### H1. 场景数 ≤3 独立空间 / 集

判定逻辑：同一独立空间（如客厅）内的所有戏份合并算 1 场，不管该空间在本集出现几次、有多少人物进出、时间是日是夜。苏家别墅的客厅、餐桌、卧室算 3 个独立空间（需 3 套画面资产），不是 1 个"苏家别墅"场景。

**扣分**：>3 场红线（强制返工）；2<场景≤3 黄线（见 S1 软规则，可过）。

### H2. 单段 OS/VO ≤3 句

OS（内心独白）/ VO（画外音）单段连续同一说话人块内不超过 3 句。超过 3 句必须拆段或改对白。[gobuildit 规范 · n=3 业界教程未量化，本数值基于《逆天鼎》观测 + 主理人决策]

### H3. OS/VO 强制带情绪标签

所有 OS/VO 段落必须用 `角色名OS(情绪):内容` / `角色名VO(情绪):内容` 格式，情绪标签在括号内。情绪词如：愤怒 / 决绝 / 忐忑 / 讥讽 / 悲怆 / 冷静 / 震惊。

原因：AI 漫剧的声优配音和表情贴图需要情绪 tag 驱动，缺情绪标签 = 配音失锚 / 表情对不上。

### H4. 单条台词 ≤6 句

漫剧分镜可承载长台词节奏（分镜切片 + 信息密度高的内容），单条台词最多 6 句话。激情爆发 / 反派威胁 / 身世交代 / 临终告别等场景可用到上限，但必须信息密度饱满，不得水话。[gobuildit 规范]

**注意与 ai_live 的差异**：ai_live 单条台词硬上限 ≤2 句（真人演员呼吸节奏约束），comic 放宽到 ≤6 句是因为分镜切片弥补。

### H5. 场景头格式 `N-N日/外或内 地点`

漫剧分场用紧凑格式 `1-1日/外叶家广场` 或 `1-2夜/内苏家客厅`（日夜 + 外内 + 地点，空格仅隔开元素组）。

**注意与 ai_live 的差异**：ai_live 场景头用 `## 1-1 · 内/外 · 地点 · 日/夜`（工业标准顺序）。comic 继承自《逆天鼎》等漫剧本子观测到的实际工业格式。

### H6. 字幕仅用于硬信息

字幕框仅出现于：品阶名 / 武技名 / 境界数字 / 人物身份 / 地名标注 / 时间跨度（"十年后"）等**硬信息**。不得用字幕承载对白、心理活动、氛围描述——那些归 OS/VO/对白层。[gobuildit 规范 · n=3 业界教程无此约束，本 skill 独有]

### H7. 开场 30 秒必爆发冲突

跨介质通用底线。详见 `opening-rules.md` 的前 30 秒钩子规则。本文件仅引用。

## 软规则（违反 = 黄线警告，可过）

### S1. 场景 ≤2 推荐节奏

gobuildit 推荐：场景 ≤2 独立空间/集（《逆天鼎》12 集抽样 100% 符合）。叙事集中度 + 资产复用 + 观众注意力聚焦的甜蜜点。

### S2. 开场 3-15 秒爆发最优

跨介质通用，前 3-15 秒出现首次冲突/爆点可最大化留存率。CSDN 数据建议 3 秒内，有戏 AI 平台 P50 45 秒——本值取中位偏激进。

### S3. 分镜数 15-25 / 集

**单位定义（必看）**：本 skill 的"分镜"采用 △ 画面描写定义——**1 个 △ = 1 个分镜 / 1 个画面单元**。

⚠️ 警示：业界存在另一种颗粒度（AI 生图 5s 镜头 ≈ 1 分镜 → 150 分镜/集）——与本规则**不兼容**。AI 漫剧团队若按 5s 镜头解读 15-25/集 规则会严重违规。

### S4. 单分镜 3-15 秒

单个 △ 画面单元覆盖的戏份时长建议 3-15 秒（CSDN 推荐 5-15s，放宽下限到 3s）。

### S5. 集时长 60-180 秒

主流漫剧集长 60-90 集 × 1-3 分钟。单集时长落在 60-180 秒区间。

### S6. △ 分镜切片密集

《逆天鼎》观测平均每 2-3 行剧本出现 1 个 △。切片越密越能承载情绪递进和节奏变化。密度偏低（>5 行/△）时给软警告。

### S7. 长台词需信息密度高

H4 允许台词 ≤6 句，但长台词必须承载：情绪爆发 / 身世交代 / 反派威胁 / 临终告别 / 关系翻转等戏剧功能。水话长台词（闲聊 / 重复 / 铺垫）违反此规则——判黄线建议拆短。

## 反 ai_live 的关键差异速查

| 维度 | ai_live | comic |
|---|---|---|
| 场景数 | 3-5 场（硬）| ≤3 硬 + ≤2 软 |
| 单条台词 | ≤2 句（硬）| ≤6 句（硬）|
| 微表情描写 | 禁用（A/B/C 三级清单）| **允许**（分镜特写可承载）|
| OS/VO 情绪标签 | 可选 | **强制** |
| 场景头格式 | `## N-N · 内/外 · 地点 · 日/夜` | `N-N日/外或内 地点` |
| 字幕用途 | 无约束 | 仅硬信息（品阶/武技/境界/身份等）|
| 对白夸张度 | 受真人质感约束 | 允许高（AI 面部弱 + 对白承担）|
