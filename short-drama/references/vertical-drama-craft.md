# Vertical Drama 剧本工艺方法论（中英文通用）

> 来源：从中文示范（叶炎复仇向 EP1）+ 英文 DWD（Dark Web Desire）transcript 抽象
> 适用：所有语言的 vertical drama 剧本（\~90 秒/集，钩子密集，移动端竖屏，走量生产+AI 配音/素人演员）
> 与 convention 的关系：本文是**工艺层**（语言无关）。convention 层（△/OS/日外/INT./V.O./SUPER 等符号）只在文末附录列对照表，不是方法论主体
> 关键坐标：本文管宏观工艺（信息密度+钩子节奏+角色发声+段落颗粒）；`dramatic-truth.md` 管微观对白戏剧真实性。两者并列加载

---

## 核心工艺原则

### 原则 1：信息单元最小化——每行一件事

**本质**：vertical drama 的一行（无论写成 `△一条动作` 还是一段 ACTION sentence）只承载**一个可拍的动作或一条可听的信息**。不做复合句、不堆两个角色的并行动作、不把「心理+环境+身份」挤进一句。原因：手机竖屏观众每秒只处理一条信号，镜头也一次只拍得下一件事。

**中文证据**（叶炎 EP1 1-1 场）：

```
△云飞雪悬浮于半空，周身圣光缭绕，眼神如视蝼蚁。
△云飞月站在一旁，嘴角挂着恶毒的笑，手中拿着一纸婚书。
△叶炎浑身是血，跪在地上，倔强地昂着头。
```

三个角色三个动作三行，不合并。一行内部只描述同一人的同步特征（悬浮+圣光+眼神共属"姿态"），不把两人动作挤一行。

**英文证据**（DWD transcript，auto-caption 颗粒已经极细）：

```
L88  Put those on.
L88  No way. I'm not going to wear it.
L89  Why am I even here?
L90  Because you're my prize that I won in a game.
```

即便是 transcript，台词也**句句断开**——没有长段独白。可反推原剧本 ACTION 段同样是每条动作一行（reelshort-style ACTION：`He throws the dress at her. She catches it. She glares.`——三个动作三句独立，不会写成 `He throws the dress and she catches it while glaring at him.`）。

**违反代价**：行颗粒一大就是"小说化"——演员/导演要二次拆，分镜密度塌陷，竖屏节奏失速。观众感受：画面慢、像在看话剧。

**可执行操作**：

- 写完一行自查：这行里有几个可拍单元？>1 就拆
- 并列两个角色的动作 → 必拆成两行
- 动作+心理+环境混写 → 心理走 OS/V.O.，环境走单独一行，动作归动作
- 连词"并/然后/and/while"出现 → 警报，大概率要拆

---

### 原则 2：每场高密度 setup——每行 0.6\~0.7 条新信息

**本质**：vertical drama 的一场（约 20 行）必须装下 **12-15 条新信息**（人设+世界观+冲突+道具+关系）。没有"氛围铺垫段"、没有"环境空镜段"、没有"留白行"。每行如果不推进信息就该删。

**中文证据**：1-1 场约 20 行装 14 条 setup（圣人等级、宗师等级、家族敌对、婚约存在、婚约被否、叶父已死、叶父恩情、民众沉默、叶炎重伤、等级规则、残剑+父魂、剑碎、绝望嘶吼、羞辱性胜利）。平均每 1.4 行一条信息。

**英文证据**（DWD EP1 区，L1-L100 约 100 个 caption-line 约等于 50 条原剧本行）装入的信息：

- L1-L5：劫持动作 + 钱损失 + 打错人（开场冲突+数字砝码）
- L7-L16：Luther 登场 + 对 Chloe 的占有宣告 + 俩兄弟有恩怨 + Chloe 是"破他心的人"（人物关系网一次交代 4 条）
- L28-L47：医生诊断 + malnourished + 性创伤 + aphrodisiac 毒品 + 子宫损伤 + 出狱 3 年背景（世界观+人物 backstory 6 条）
- L48-L60：Luther 兄弟对话 + 毒品调查指令 + 命令互抗（权力结构 3 条）

约 50 条原剧本行装 13-15 条新信息。密度与中文示范持平。

**违反代价**：密度稀 → 观众划走。Reelshort 平台数据反复验证："前 30 秒没装进 3 条钩子+2 条关系+1 条世界观规则"的剧集完播率断崖。

**可执行操作**：

- 写完一场列 setup 清单：这场交代了哪几条新信息？<10 条 → 密度不够
- 每行自查：这行提供了什么新信息？提供不了 → 删或合并前后行
- 禁止写"氛围戏"（纯环境描写、纯心理反刍而无新信息）

---

### 原则 3：节点密度——每场 2+ 个剧情节点，集尾硬钩

**本质**：vertical drama 一集（约 90 秒 / 2 场 / 40 行）必须推进 **4-5 个剧情节点**，即场均 2-2.5 个节点。每集必须以**事件钩子**结尾（下一秒会发生什么），不是**情绪钩子**（下一秒他会怎么想）。

**中文证据**：叶炎 EP1 两场戏 5 个节点——① 公开羞辱 ② 父剑被碎+婚书被撕 ③ 复仇誓言 ④ 帝体觉醒 ⑤ 齐家压门。集尾是**齐家打上门要砸圣碑**——敌人明确、时间压力明确、物理冲突立刻发生。

**英文证据**（DWD 节点追踪，samples）：

- L60-L65 场尾："Contact the prison. Look into the drug. / Look into Chloe Morgan." → 事件钩子（调查启动，新信息要爆）
- L161 场尾："Liam, where's Chloe? Hunt her down." → 事件钩子（追捕令，下一集必冲突）
- L213-L215 场尾："Put her in her secret chamber. Don't let her out until I say so." → 事件钩子（物理囚禁，状态升级）
- L535-L538 场尾："Chloe, she escaped. / She promised she wouldn't run away again." → 事件钩子（逃走+背叛悬念）

每个钩子都是**某人立刻要做某事**，不是"他陷入沉思"。

**违反代价**：情绪结尾 = 观众无下集驱动力 = 完播率掉。"他望着窗外想起往事"这类结尾在 vertical drama 里 = 死刑。

**可执行操作**：

- 集尾写完自查：下一秒会发生什么具体事件？答不出 → 重写
- 钩子模板选一：新敌登场 / 物理威胁 / 身份反转揭露 / 关键道具易手 / 时间压力（"X 分钟后 Y 就会 Z"）
- 禁用：主角发呆/沉思/流泪/回忆/独处拉长镜作结

### 原则 4：零废话台词——每句台词必承担情绪峰值或信息投喂

**本质**：vertical drama 的台词没有"日常口语"、没有"寒暄铺垫"、没有"温情闲聊"。每句台词要么是情绪冲顶（怒吼/威胁/羞辱/誓言），要么是关键信息投喂（世界观规则/身份揭露/历史交代/道具设定）。**没有第三种**。

**中文证据**：叶炎 EP1 台词全部是情绪峰值或世界观投喂——

```
云飞雪：圣人之下，皆为蝼蚁。本圣之言，便是证据。  [世界观规则 + 等级宣告]
叶炎：无耻！当年我父叶啸天与云老家主亲手订立婚约……  [历史交代 + 怒吼情绪]
云飞月：你也不撒泡尿照照自己……  [羞辱性挑衅 + 情绪冲顶]
```

无一句"今天天气不错"、"你吃了吗"级别的过渡台词。

**英文证据**（DWD）：

```
L59   You are mine. Just do as I say.                   [占有宣告 + 权力情绪]
L89   Because you're my prize that I won in a game.     [世界观规则：女人是赌局战利品]
L115  Let's drown ourselves in desire.                  [诱惑+人设词汇]
L138  I'd rather keep you by my side and watch you die a slow death.  [报复誓言情绪]
L598  I promise to take care of you.                    [反转情绪峰值]
L1168 From now on, I will carry the darkness on my shoulders so that you may live in your light.  [婚礼誓言 + 关系反转]
```

没有 "Hi, how are you" / "Nice weather" / "Let me explain what I mean" 级别的缓冲台词。

**违反代价**：一旦出现日常口语 → 观众察觉节奏掉挡 → 划走。写手容易犯：为"写得自然"加过渡句，结果把紧绷感泄掉。

**可执行操作**：

- 每句台词自查：这句话在干什么？只有"情绪峰值"、"信息投喂"、"世界观设定"三类合格，其他一律删
- 禁用台词清单：寒暄、重复确认（"你说什么？""我说……"）、客套、温情闲聊、过渡解释
- 信息和情绪可合并（叶炎"无耻！当年我父……"一句做两件事），越密越好

### 原则 5：台词必须带情绪/动作标注——演员不留二次发挥空间

**本质**：vertical drama 的每句台词几乎都紧跟一个**情绪词或动作词**作为表演指令。原因：vertical drama 是流水线生产（几天拍 80 集）+ AI 配音 + 素人/低价演员，不能靠演员"自己体会人物"，情绪必须写死在剧本层。

**中文证据**：

```
云飞月 (甩下婚书)：叶炎！这婚书是假的！……
叶炎 (怒吼)：无耻！……
云飞雪 (眼神一寒)：冥顽不灵。
云飞月 (一惊，随即讥笑)：哟？你叶家真是没落了……
```

EP1 台词几乎 100% 带 paren 标注，分三类：纯情绪（怒吼/震惊/狂喜）、纯动作（甩下婚书）、情绪+动作复合（一惊随即讥笑）。

**英文证据**（transcript 没 paren，但台词本身带**情绪动词**或**动作暗示**承担同样功能）：

```
L14   Let me go, you creepy monster.              [动词"let go"+人设词"creepy monster"=惊恐挣扎]
L20   Holy  The great sea interested in Cleopatra.  [反讽修辞=戏谑嘲弄情绪]
L89   Put those on. / No way. I'm not going to wear it.  [命令+拒绝节奏=对抗]
L147  Who the hell do you think you are?          [挑衅句式=愤怒]
```

Transcript 反推：英文 vertical drama 原剧本同样要求每句台词有 `(parenthetical)` 缩进指令。Reelshort/ShortMax 等平台剧本模板都强制要求——这是 AI 配音和素人演员能出活的前提。

**违反代价**：不写情绪 → 配音平淡/演员出戏 → 观众感受"像念台词"。AI 配音场景下尤其致命：TTS 需要明确情绪 tag 才能调语调。

**可执行操作**：

- 每句台词后必须有 (情绪) 或 (动作) 标注，**除非**台词本身句式已经强制了情绪（"Let me go!" "Stop!" 感叹句）——这种可省略
- 情绪词选择：具象、可演（"怒吼"/"森寒"/"讥笑"/"颤抖" good；"复杂"/"五味杂陈"/"深沉" bad，不可演）
- 允许复合标注（"一惊，随即讥笑"），标注情绪转折

### 原则 6：内心独白专用于"必须角色主观说出"的稀缺节点

**本质**：OS（中文）/ V.O.（英文 voiceover）/ O.S.（off-screen）只在**觉醒瞬间 + 认知突变 + 必须角色脑内才能说清的世界观信息**三类场景使用。不做日常叙事、不做情绪反刍、不做旁白补丁。

**中文证据**：叶炎 EP1 全集仅 2 条 OS，都在 1-2 场**帝体觉醒瞬间**：

```
叶炎 OS(震惊)：我的经脉……在重塑？这股力量……
叶炎 OS(狂喜)：金色丹田！万古仅有！父亲曾说这是传说中的万道帝体……
```

功能：把"金色丹田=万古仅有""万道帝体=传说设定""封印破碎=觉醒机制"三条核心世界观信息一次性塞进角色脑内——不写旁白、不写字幕长注解、不逼角色对空气说话。

**英文证据**（DWD 的 V.O. 化用法——通过"诗化 monologue"承担同样功能）：

```
L504-L508  "How you walked on fires of hell / and lured angels into the abyss. / The universe trembles and shifts. / The currents of history erode our world. / And finally, we're availing it in darkness."
L561-L562  "Rebirth is painful, / but we must endure to start a new."
L614-L619  "Death is so final. / Whereas life is full of possibilities. / If I were you, / I'd live to make Shawn pay. / Don't look so gloomy. / Remember, you're free now."
```

这些是情绪/认知突变点的**诗化 V.O.**——Sean 和 Carl 在关键转折（Chloe 重生 / 流产后 / 追杀发动）给出主观认知宣告，不承担日常叙事，只承担世界观+人物内心法则的压缩投喂。

**违反代价**：OS/V.O. 滥用 → 剧本变成"旁白推进剧情"→ 画面失效 → 廉价感爆炸。最常见病：用 OS 写角色在日常场景的心情反刍。

**可执行操作**：

- 一集 OS/V.O. ≤ 2 条
- OS 触发条件三选一：① 觉醒/认知突变瞬间 ② 必须角色主观说出的世界观规则 ③ 重大决策的内心最终宣告
- 禁用：日常心情反刍（"他好冷漠""我好难过"）、叙事补丁（"他走到了门口"）、情绪重复（台词已经怒吼了再 OS 一遍"我真的很气"）

### 原则 7：反派威胁 = 话语+破坏性动作绑定；主角 underdog = 身体挫败+精神不屈+情感锚

**本质**：vertical drama 的角色塑造在剧本层就写死——**反派每说一句狠话必须跟一个破坏性动作**（不让反派空口威胁），**主角每次被打击必须四件套齐全**（身体挫败 + 精神不屈 + 情感锚 + 主观发声）。这不是演员表演决定，是剧本结构决定。

**中文证据**（反派绑定）：

```
云飞雪："蝼蚁"                → △ 抬手一指，圣威压下
云飞雪："碎！"                → △ 屈指一弹，剑炸裂
云飞月："废物"                → △ 撕碎婚书，碎片洒脸
```

**中文证据**（主角四件套）：

```
身体挫败：△叶炎浑身是血，跪在地上
精神不屈：倔强地昂着头 / 拼尽全力想要站起
情感锚：  父亲的残剑+叶啸天虚影显形
主观发声：叶炎OS"此仇不报我叶炎誓不为人"
```

**英文证据**（反派绑定，DWD）：

```
L59    "You are mine. Just do as I say."     → L88 物理动作：throws dress at her（"Put those on"）
L108   "One step closer and I'll knock you out."  → 紧接 L111-L115 Carl 的强迫诱惑动作
L214   "Put her in her secret chamber."      → L215 物理囚禁动作执行
L665   "Even if it was her idea, it was you who did it." → L673-L674 情绪/指控+随后流产事件爆发
```

**英文证据**（主角四件套，Chloe）：

```
身体挫败：L29 malnourished / great emotional trauma / sexual assault history
精神不屈：L14 "Let me go, you creepy monster" / L116 "Not in a million years"
情感锚：  L150-L159 Jane（好友）出现 + Lily（妹妹）死因悬念
主观发声：L622-L623 "Since I can't escape from hell, I might as well embrace it"（认知宣告）
```

**违反代价**：反派空口狠话 → 观众觉得"嘴炮"→ 威胁感崩。主角只有挫败没有不屈 → 观众放弃共情（pathetic 不等于 underdog）。

**可执行操作**：

- 反派每句威胁语写完立刻自查：后面跟了什么破坏性动作？没有 → 补
- 主角每个挫败段落自查：四件套齐了吗？缺哪件？补
- 反派可以双人分工互补（冷威型 + 毒辣型 / 权力型 + 挑衅型），一个建威慑一个建激怒

### 原则 8：场间硬切不解释过渡；镜头语言隐身

**本质**：vertical drama 场与场之间**硬切**——不写"他被抬回家"的中间戏、不写"镜头淡出"的过渡、不写"时间来到三天后"的字幕补丁（除非是大跳跃）。剧本**不写镜头术语**（不写中景/特写/全景/CLOSE-UP/MEDIUM），让动作描述本身暗示景别。镜头决策权归导演，编剧只负责信息打包。

**中文证据**：叶炎 EP1 从 1-1 广场 → 1-2 叶炎房间，无过渡段。叶炎怎么从广场回到房间？不解释。全剧无"中景""特写"等镜头术语，只有 △动作。

**英文证据**（DWD 场切节奏）：

- L63 "Let me know what you find." → L64 直接跳到 "Look into a woman named Chloe Morgan."（不同场景 / 不同对话者 / 无过渡）
- L83 "The Luther family would never accept a woman like you." → L85 直接跳到医生诊室："I'm sorry, Miss Stone, but with your condition, it's impossible..."（场景跳、人物跳、时间跳，无解释）
- L150 "Jane. Chloe. I thought you were in prison." → 前一场 Chloe 还在被囚，这场直接见到好友，不解释怎么见上的

**违反代价**：过渡戏多 → 单集节奏废行多 → 在 \~90 秒/集的约束下直接挤掉有效剧情节点空间。

**可执行操作**：

- 场切之间禁写"他走出门""时间来到""镜头切到"之类过渡
- 重要时间跳用一条字幕（中文 `△字幕：三天后` / 英文 `SUPER: THREE DAYS LATER`），一行解决
- 剧本里不写景别术语，用动作描述隐式传达（"叶炎眼神一寒"暗示特写；"云飞雪悬浮半空"暗示全景）——导演自决

## 分元素工艺标准（每类元素的密度 / 颗粒度 / 节奏）

> 本节是上一节 8 条整体原则的**元素分解**。整体原则告诉你"剧本节奏要怎样"，本节告诉你"具体每类元素正常可用的标准是什么"。Polish Agent 写或改某一类元素时优先查本节。
>
> 数字来源：叶炎 EP1 1-1 / 1-2 两场戏实测（37 行，10 条△ + 9 条△ + 13 条台词 + 2 条 OS），DWD EP1 段 L1-L160 实测（\~50 条原剧本行等价）。所有数字都是样本反推的真实值，不是理论推测。

### A. 人物台词（Dialogue）

**长度标准**：

- **中文**：单句（以句号/感叹号/问号断）**≤25 字**；单行台词可以用多子句串联但**总字数 ≤90 字**（仅情绪峰值句可到此上限，如叶炎怒吼 92 字列家族恩怨，是全集最长台词）；常规台词单行 **30-60 字**
- **英文**：单句 **≤15 words**；单行台词（一个 character cue 下的一个 speech block）**≤30 words**；威胁/誓言类情绪峰值句可到 30 words，常规对抗句 6-15 words
- 中文样本实测：13 条台词中 11 条单句 ≤20 字，最短 2 字（"不——！"），最长 92 字（怒吼情绪峰值）
- 英文样本实测：DWD L1-L160 的 \~40 句台词 95% 单句 ≤12 words，L59/L90/L138 等峰值句 10-18 words

**密度标准**：

- 一场戏（20 行）**台词条数 6-8 条**，占全场行数 **30-40%**
- 单角色连续台词\*\*≤2 条\*\*——第 3 条必须由对方角色打断或由 △动作切走（叶炎怒吼后云飞雪立刻接，绝无主角连续 3 句独白）
- 非情绪峰值场景（调查、信息传递类）台词可到 50%；情绪峰值场景（公开羞辱、对质）台词 30% + 动作 60%

**功能标准**（每句必承担其中之一，见原则 4）：

- 情绪冲顶（怒吼、威胁、羞辱、誓言）
- 关键信息投喂（世界观规则、身份揭露、历史交代、道具设定）
- 钩子转向（集尾/场尾的"下一秒要发生什么"指令）

**禁止类型**（举反例）：

- ❌ 寒暄："你来了。" / "早啊。" / "Hi, how are you."
- ❌ 复述对方台词确认："你说婚书是假的？" / "You mean the USB?" —— 观众刚听过，不用二次确认
- ❌ 独白堆叠（>2 条连续同角色台词填情绪）："我怎么会这样。……为什么是我。……命运太不公。" —— 改走 OS 1 条或动作表达
- ❌ 过渡解释："我先走了" / "稍等一下" / "Let me explain." —— 场切硬切，不解释
- ❌ 客套温情："谢谢你" / "你真好" —— vertical drama 没有温情缓冲
- ❌ 凑节奏的日常对白："今天天气不错" / "Nice weather"

**中文证据**（1-1 场）：

```
云飞雪："蝼蚁之物，也配显形？碎！"（13 字，情绪+命令单句）
叶炎："不——！"（2 字，情绪峰值）
```

短句极限与情绪爆点绑定。

**英文证据**（DWD）：

```
L59   You are mine. Just do as I say.              （8 words, 占有宣告）
L116  Not in a million years.                      （5 words, 拒绝峰值）
L138  I'd rather keep you by my side and watch you die a slow death.  （15 words, 报复誓言）
```

### B. 动作描述（Action / 中文 △）

**长度标准**：

- **中文**：单条 △ **15-25 字**为正常区间；上限 **35 字**（仅用于"因果链必须连写"的情况，如云飞雪放招→叶炎受击的一条 △ 31 字）；≥40 字必拆
- **英文**：单条 ACTION sentence **6-15 words**；复合因果可到 20 words
- 中文样本实测：19 条 △ 平均 22 字，范围 15-36 字。36 字的那条（"叶炎全身骨骼发出爆响……动弹不得"）是极限，已经到拆分临界

**密度标准**：

- 一场戏（20 行）**△条数 8-12 条**，占全场行数 **50-65%**
- 动作 : 台词比例区间 **1.5:1 \~ 2:1**（动作是主要载体，不是辅助）
- 情绪峰值场景动作可到 70%，调查/对话场景动作可降到 40%

**颗粒度标准**：

- **必写**：关键道具操作（撕婚书、碎剑、抬手放招）、位置变化（悬浮半空、跪在地上、翻身下床）、视觉爆点（金光爆发、圣威压下、血渗毛孔）
- **禁写**：演员下意识小动作（挠头、摸鼻子、眨眼）、无关环境（窗外的鸟叫、桌上的茶杯）、抒情化动作（"她优雅地转身"/"风轻轻拂过她的发梢"）

**一条 △ 承担的事件数上限**：1 个主动作（who does what）+ 可选 1 个同步状态（同一人的伴随状态，如"抬手，一指点出"算 1 个动作两段）。**禁止**：两人并行动作挤一条 △（必拆两行）

**动作 vs 镜头术语的边界**：

- 够暗示景别的（不写镜头词）：「叶炎眼神一寒」（隐含特写）/ 「云飞雪悬浮于半空」（隐含全景）/ 「金光瞬间蔓延至全身，透过皮肤，将昏暗的房间照得通亮」（隐含从特写拉到中景）
- 真的不行要硬写镜头的（极罕见）：涉及主观 POV 镜头切换、or 必须明确蒙太奇的（如闪回），此时可用 △ 字幕式备注（`△闪回：三年前`），但仍**不写** `CLOSE-UP/MEDIUM SHOT` 等术语

**禁止类型**（举反例）：

- ❌ 两人挤一条："△云飞雪抬手压下圣威，叶炎骨骼爆响口吐鲜血" —— 必拆两行
- ❌ 抒情化："△叶炎心如刀绞地望着碎剑，泪水在眼眶里打转却倔强地不肯落下" —— 心理走 OS 或情绪词，抒情不是动作
- ❌ 无关环境："△广场上的石板被岁月磨得光滑，远处古树的枝叶在风中摇曳" —— 与情节无关，删
- ❌ 演员下意识小动作："△叶炎紧张地挠了挠头，吞了吞口水" —— 不可拍或不用拍

**中文证据**（1-2 场）：

```
△金光瞬间蔓延至全身，透过皮肤，将昏暗的房间照得通亮。（25 字，视觉爆点）
△叶炎原本断裂的经脉，在金光的冲刷下，以肉眼可见的速度重塑、连接。（31 字，因果链必须连写，临界长度）
```

**英文证据**（DWD 反推）：

```
L88 动作链反推："He throws the dress at her. / She catches it. / She glares."
三个动作三句，最小颗粒。不会写成"He throws the dress which she catches while glaring."
```

### C. 表情/神态（Reaction / Expression）

**判定标准（独立成行 vs 嵌入 paren）**：

- **嵌入 paren**（默认，\~80% 情况）：伴随台词的情绪，写在 `(情绪)` 里——"（怒吼）"、"（森寒）"、"（一惊，随即讥笑）"
- **独立成行**（保留给情绪爆点 + 无台词时刻）：反派放杀招前的冷眼、主角受击后的绝望、觉醒瞬间的震惊——此时写作独立 △ 条："△云飞雪眼神一寒"、"△叶炎双目充血"

**颗粒度标准**：

- ✅ **够用**：具象、可演、单词级情绪词——"眯眼"/"冷笑"/"眼神一寒"/"咬牙"/"面无表情"/"瞳孔骤缩"/"glares"/"smirks"/"eyes narrow"
- ❌ **过粗**（抽象不可演）："复杂的情绪" / "五味杂陈" / "难以名状的神色" / "百感交集" / "a complicated expression" / "mixed feelings"
- ❌ **过细**（文学化、演员做不到）："眉头微蹙至 0.3 厘米" / "嘴角上扬了一个微小的弧度" / "a barely perceptible 2mm twitch of the left eyebrow"
- ❌ **模糊指令**（把活丢给演员）："陷入沉思" / "若有所思" / "情绪复杂" / "deep in thought" —— 不知道该演什么

**密度标准**：

- 一场戏神态独立行 **≤3 条**（约占全场 10-15%），paren 内嵌情绪词 **每条台词 1 个**（频次不限，因为本身紧贴台词）
- 反派威胁场景神态密度可到 4-5 条（每次放招前都给冷眼、轻蔑、森寒的独立条）——反派需要靠表情**建威胁感**
- 主角受击场景神态密度 2-3 条——主角靠"不屈四件套"（身体挫败+精神不屈+情感锚+主观发声），不是靠堆表情

**反派 vs 主角表情写法差异**：

- 反派：**主动型、破坏性绑定**——"眼神一寒"后必跟"抬手放招"（原则 7）；表情是威胁的预警信号
- 主角：**被动型、不屈型**——"双目充血"但"拼尽全力想要站起"；表情展示挫败+不屈的张力

**中文证据**（1-1 场）：

```
云飞月 (一惊，随即讥笑)：哟？…… （paren 嵌入，情绪转折标注）
△叶炎双目充血，拼尽全力想要站起。 （独立行，主角受击不屈）
云飞雪 (眼神一寒)：冥顽不灵。    （paren 嵌入，反派放招预警）
```

**英文证据**（DWD transcript 反推对应原剧本 paren）：

```
L108  "One step closer and I'll knock you out."
  → 反推 paren：(glares) 或 (threatening)，而非 (feeling complex emotions)
L147  "Who the hell do you think you are?"
  → 反推 paren：(fury rising) 或 (snaps)，单词级情绪
```

### D. 场景描述（Setting / Establishing）

**允许行数**：

- 场标头后的 setup 行：**开场或新场景首次出现 ≤5 行** △，覆盖（在场人物姿态 + 关键道具 + 必要空间关系 + 必要氛围标记）
- 中段回到同一场景：**0 行**环境描述，直接进动作/台词
- 同场内的后续 △：只写推进情节的动作，不再描述环境

**必写元素**：

- 时间/地点（已在场标头）
- 影响后续动作的光线/空间（"昏暗的房间"——因为后面有"金光将房间照得通亮"的对比）
- 关键道具的初始状态（"手中拿着一纸婚书"——后面要被撕）
- 在场人物的初始位置/姿态（悬浮、跪地、站一旁）
- 群众/背景角色的情绪反应（"民众指指点点，神情鄙夷"——建立羞辱氛围的社会压力）

**禁止元素**：

- 与情节无关的环境（地板材质、窗外风景、家具细节）
- 抒情化环境描写（"夕阳的余晖温柔地洒在广场上"）
- 气味/触感等非视觉描写（vertical drama 不需要文学通感）
- 对空间做文学化修辞（"这是一个被时间遗忘的角落"）

**人物入场写法**：

- 场标头下一行写 `人物：A, B, C, 群众` 列清单
- 人物**初次入场**用独立 △ 描述姿态/位置（5 行 setup 内）
- 后续入场（如叶炎 1-2 场末家仆闯入）用动作条 △ 带出："△房门突然被'砰'地一声撞开，一名家仆惊慌失措地冲进来。"——入场 = 动作，不是环境

**道具写法**：

- **关键道具**（推动情节的）：首次出现明确写（"手中拿着一纸婚书"），后续每次操作**必须再写**（"随手撕碎婚书" / "碎片洒在叶炎脸上" / "叶炎躺在床上，看到身前碎裂的天帝剑" / "叶炎伸手去抓断剑碎片"）
- **背景道具**（装饰性的）：**不写**
- USB / 婚书 / 剑 / 毒品 / 项链这类 callback 道具必须每次出现时给一条 △ 锚定

**场景切换过渡选择**：

- **硬切**（默认）：场号直接跳（1-1 → 1-2），不写"他被抬回家"。叶炎 EP1 从广场到房间就是硬切
- **软切**（仅用于同一时间同一动作延续）：前一场最后一条 △ 是动作（如"冲出门外"），下一场第一条 △ 接续该动作（"叶炎推开大门"）——只在节奏需要连贯时用
- **字幕过渡**（大时间跳）：`△字幕：三天后` / `SUPER: THREE DAYS LATER`，一行解决

**中文证据**（1-1 场开场 5 行 setup）：

```
△云飞雪悬浮于半空，周身圣光缭绕，眼神如视蝼蚁。
△云飞月站在一旁，嘴角挂着恶毒的笑，手中拿着一纸婚书。
△字幕：云氏家族圣人品阶云飞雪、一品宗师云飞月。
△叶炎浑身是血，跪在地上，倔强地昂着头。
△周围是密密麻麻的苍云城民众，指指点点，神情鄙夷。
```

5 行覆盖：3 个主角姿态 + 关键道具（婚书）+ 身份字幕 + 群众压力。之后不再写环境。

**英文证据**（DWD 场切节奏）：

```
L83  "The Luther family would never accept a woman like you."
L85  "I'm sorry, Miss Stone, but with your condition, it's impossible..."
```

从 Jane 的台词直接硬切到医生诊室，无 transitional action、无"Later at the clinic"过渡说明。

### E. 内心独白（OS / V.O.）

**一集次数上限**：**≤2 条**。叶炎 EP1 用满 2 条（震惊+狂喜，同一个觉醒瞬间的连续 2 条可算 1 次使用）；DWD 英文版用"诗化 V.O."承担（一集 2-3 条段落）

**触发条件**（三选一）：

1. **觉醒/认知突变瞬间**：主角突然获得新能力或理解新真相，需要内部宣告（叶炎"金色丹田！万古仅有！"）
2. **必须主观才能说出的世界观规则**：用旁白或字幕会廉价，用对话会强塞（"父亲曾说这是传说中的万道帝体……原来封印破碎，便是帝体觉醒之时"—— 世界观 + 人物记忆 + 因果链一次性压缩）
3. **重大决策的内心最终宣告**：DWD L622-L623 "Since I can't escape from hell, I might as well embrace it." —— 主角转向的认知拐点

**禁止替代**：

- ❌ 动作可以表达的不用 OS：叶炎翻身下床一拳爆鸣 = 表达"力量恢复"，不用 OS "我力量回来了"
- ❌ 对手能直接说的不用 OS：反派威胁用台词说，不用主角 OS "他在威胁我"
- ❌ 日常心情反刍："我好难过" / "真不敢相信" / "他怎么能这样"
- ❌ 叙事补丁（用 OS 解释剧情）："他走到了门口"（画面已经演了）

**单条字数上限**：

- **中文**：单条 OS **≤50 字**。叶炎 OS1 = 14 字（震惊瞬间短促），OS2 = 42 字（觉醒信息量大，接近上限）。>50 字必拆或改走动作
- **英文**：单条 V.O. **≤25 words**。DWD L504-L508 "How you walked on fires of hell / and lured angels into the abyss..." 是诗化 V.O.，5 短句合计 \~25 words，是上限

**OS 与画面动作的同步关系**：

- OS 念完 = 一个动作或一个视觉爆点完成，不能让 OS 独自跑两三个镜头（画面会空）
- 叶炎 OS1（震惊）配 △经脉重塑；OS2（狂喜）配 △骨骼愈合+翻身下床——**OS 永远伴随可见的画面变化**，不单飞

**中文证据**（1-2 场觉醒）：

```
△断裂的骨骼迅速愈合，发出噼啪的爆鸣声。
叶炎 OS(震惊)：我的经脉……在重塑？这股力量……
叶炎 OS(狂喜)：金色丹田！万古仅有！父亲曾说这是传说中的万道帝体……原来封印破碎，便是帝体觉醒之时！
△叶炎翻身下床，动作轻盈矫健，一拳挥出，空气发出一声爆鸣。
```

两条 OS 夹在两条 △ 之间，画面持续变化。

**英文证据**（DWD 诗化 V.O.）：

```
L561-L562  "Rebirth is painful, / but we must endure to start a new."
（Chloe 流产 + 重生转向的认知宣告，2 短句 ~10 words，配同步的死后重生画面）
```

### 元素配比合理区间（一场戏内，20 行基准）

| 元素        | 占行比     | 条数     | 触发场景                    |
| --------- | ------- | ------ | ----------------------- |
| 动作描述 △    | 50-65%  | 8-12 条 | 全程（vertical drama 主载体）  |
| 台词        | 30-40%  | 6-8 条  | 对质/羞辱/威胁/信息投喂           |
| 神态独立行     | ≤15%    | ≤3 条   | 反派放招预警、主角受击不屈、觉醒瞬间      |
| 场景描述      | 仅场首 5 行 | 0-5 条  | 开场或新场景首次出现，中段回到同场景写 0 条 |
| OS / V.O. | ≤5%     | ≤2 条/集 | 觉醒、认知突变、世界观主观投喂         |

**配比自查**：

- 动作 < 50% → 对话太多，像话剧，不像 vertical drama
- 动作 > 70% → 可能是纯打戏，缺少台词投喂信息，setup 密度不够
- 神态独立行 > 3 条 → 可能在用表情替代动作推进，查一下是否该改写成动作
- 场景描述 > 5 行 → 小说化，删掉与情节无关的环境
- OS > 2 条/集 → 用 OS 偷懒了，查哪些可以改走动作或对话

## 综合：vertical drama 工艺的 1 条最高原则

**如果只能记一条——「每条内容单元（一行动作 / 一句台词 / 一个镜头）都必须在信息密度或情绪峰值上承重，否则立刻删」。**

原则 1-8 都是这一条的展开：

- 每行一件事 = 单元颗粒最小 → 才能密集装
- 每场 12+ setup = 信息承重下限
- 节点密度 = 剧情承重下限
- 零废话台词 = 台词承重下限
- 情绪标注 = 表演层承重
- OS 稀缺 = 把稀缺通道留给真正的承重点
- 反派绑定+主角四件套 = 角色塑造承重
- 硬切 = 过渡段不承重 → 删

vertical drama 是**承重测试压到极致**的格式。写普通剧本时 20% 的台词/动作可以是"节奏/氛围/留白"，在 vertical drama 里这 20% 全部要删到 0。

## 附录 B：与 dramatic-truth.md 的关系

**本文（vertical-drama-craft.md）**：宏观工艺层

- 管：信息密度、节点密度、钩子节奏、单元颗粒、角色发声差异化、场切节奏、OS 使用时机
- 单位：行、场、集

**dramatic-truth.md**：微观对白工艺层

- 管：单句台词的戏剧真实性、次文本、对白的潜台词和冲突层级
- 单位：句

**加载顺序**：写剧本时两者并列加载——先按本文保证宏观结构（密度/节点/钩子），再按 dramatic-truth 精修每句台词的戏剧真实性。两者不冲突：本文要求"每句台词承担情绪峰值或信息投喂"（原则 4），dramatic-truth 要求"这句台词不能是直白告知，要有潜台词"——前者管"要不要存在"，后者管"怎么写才活"。

**冲突判断**：如果 dramatic-truth 的某条建议会导致本文原则 2/3/4 的密度或节点受损（例如"为了潜台词留白多加两行铺垫"），以本文为准——vertical drama 的格式约束是前置硬框架，戏剧真实性服从于节奏。
