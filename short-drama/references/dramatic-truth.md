# Dramatic Truth — 对白工艺反 Trailer-Speak 硬约束

> **本文件定位**：与 `hard-rules.md` / `anti-patterns.md` / `dialogue-craft.md` / `platform-knowledge.md` / `script-element-extraction.md` 并列的 Script Agent / Polish Agent / Audit 4 强制 reference。
>
> **触发场景**：写或改任何角色长台词（≥10 词的非 fragment 句）时强制加载。Audit 4 戏剧真实性专测使用本文件 4 症状清单。
>
> **来源**：2026-04-15 外部审查 vertical drama demo EP1-3 初稿，给出"主角不像活人"4 症状诊断 + 4 味药方。本工艺约束适用全 skill，不限单一 demo。

## 病根：执行符合设计 ≠ 戏剧真实

3 层 audit（结构/voice/借鉴）查的是"是否符合 Bible + 骨架 + references"，**没查"台词在场景内是否像戏内人物对戏内对手说话"**。Bible §15.5 v1 和对标剧（Dark Web / ReelShort）本身带 Trailer-Speak 病——把"长句 + 排比 + 比喻 + 替观众总结"当作"voice 强"。Audit 把"执行符合设计"标 PASS = 把病一路放行。

**戏剧真实性的判定基准**：把这句台词读给戏内对手角色听，他/她会觉得：
- 这是给我说的吗？还是给镜头外观众说的？（→ As-You-Know-Bob）
- 这句话里有信息，还是有花活？（→ Metaphor Overdose）
- 这句话像剧本，还是像人在这一刻能说出来的？（→ Trailer-Speak）
- 说这句话的人剩 6 个月命，他真会停下来念这一段吗？（→ Urgency Mismatch）

任一答错 = 这句不是戏剧真实，必须重写。

## Gemini 4 症状清单（Audit 4 grep checklist）

### 症状 1：Trailer-Speak（预告片体）

**表征**：句子写得像预告片字幕——平行三段式、时间倒计时排比、戏剧化停顿。

**典型 pattern grep**：
- `三段平行`：`X，X。X，X。X，X。` 同结构 ≥3 次（如"三年，我擦地板。三年，是、先生。三年，我留着底。"）
- `倒计时排比`：`今晚——XX 秒——XX` / `XX 年——XX 秒——XX` 破折号倒计时
- `数词排比`：`一张X。一辆X。三年前。` 报菜名式数词起头
- `单字句号堆叠`：`副本。每一间屋子。每一个名字。每一夜。` 名词独立成句 ≥3 个连排

**判定**：grep 命中 = Trailer-Speak。EP1-3 重写后必须 0 命中。

**为什么是病**：预告片字幕的功能是 30 秒卖整剧——压缩、修辞化、对观众喊话。剧本对白的功能是戏内人物在这一刻对这一对手说这一句。两者目的相反。预告片体放进剧本 = 戏内对手必须装听不懂、观众觉得"又在演了"。

### 症状 2：Metaphor Overdose（比喻超载）

**表征**：用比喻替代事实；单场景比喻数量超过 1 个；比喻系统化（屋子/裂缝/底色/墙/血 一组连用）。

**典型违规**：
- "我不要了" → 改成 "我留着底" → "三年我擦他们地板上的血" → "你不进的屋子"（4 个比喻一场戏）
- 学霸的狠是**定义精准**（"威胁的词根 minari 意味着你还有底牌"）不是**比喻繁复**（"屋子/裂缝/底色"）

**硬规则**：
- 单场景比喻 ≤1 个
- 比喻必须能砍掉而句意不破——能砍 = 装饰（OK），不能砍 = 用比喻承载信息（违规，重写为定义/事实）

### 症状 3：As-You-Know-Bob（背景信息倾倒）

**表征**：角色对一个本就知道这件事的对手说出这件事——名义上是对话，实际上是对观众解释背景。

**典型违规**：
- ep001:379 Sarah 对 Don Valenti 说"三年，我站在你不进的屋子里" → Don Valenti 是雇主，他比谁都知道她干了三年。这句是对观众说的。
- ep001:381 Sarah 对全厅说"副本。我擦过的每一间屋子。每一个名字。每一夜。" → 厅里坐的都是知情人。这是替观众做剧情总结。

**判定测试**（每句 Sarah 长台词必问）：
> 把这句读给戏内对手角色听——他会觉得 (a) 这是新信息 / (b) 我知道，你想说什么？

(b) = As-You-Know-Bob，重写。重写方向：要么删掉（戏内已知不需要说），要么转为**操作动作**（不说"我有副本"，把 USB 放桌上让对方看到）。

### 症状 5：Body-Mute（身体在地狱、台词在课堂）

**表征**：角色处于生理失控状态（剧痛 / 戒断 / 流产 / 溺水 / 极度震惊 / 刚听到死亡宣判 / 第一次见到尸体 / 被掐脖子）时，台词仍是结构完整的句子——**身体没接管语言**。

**典型违规判据**：把这场戏的台词从场景里抽出来，单独读——能看出说话人正在发生身体危机吗？看不出 = Body-Mute。

**对照参考**（**描述**，不是规范）：身体接管语言时，自然语言会出现以下任意几样：
- 重复（"Please. Please. Please."）
- 单词独立成句、未完成句、卡住的句首
- 气音 / 倒抽 / 半句吞回去
- 动作语言压过命题语言（"它——它在动" 比 "婴儿还活着" 自然）
- 称呼变形（叫错名字、退回到 mom/dad、用全名）

**为什么是病**：观众的**镜像神经元**会读演员的身体。如果演员演了气短抽搐但台词是 "I have come to a determination concerning the matter"——观众的身体读到一个信号，耳朵听到另一个，**出戏**。Voice contract 把"角色永远精准"当作美德，会把所有生理失控场都写成"角色精准地描述自己的失控"——这是文学，不是戏剧。

**与症状 4 (Urgency Mismatch) 的区别**：
- 症状 4 关心**紧迫场景下台词是否塞了不必要的修辞**（应该砍铺垫直奔动作）
- 症状 5 关心**生理失控场景下台词是否仍然结构完整**（应该允许散裂、重复、未完成）
- 一个角色 6 个月命但坐在咖啡馆 → 触发症状 4 不触发 5；一个角色刚被宣判 6 个月命当场说话 → 同时触发 4 和 5

### 症状 4：Urgency Mismatch（紧迫感错配）

**表征**：剩 6 个月命/正被追杀/30 秒后要死的角色在念结构整齐的段落。

**典型违规**：
- ep003:263 Sarah 6 个月命 + 在敌方车里 + 第一次正面递核弹证据 → "一张我不该发现的照片。一辆撞毁的车。三年前。" 三段式报菜名 = 没有任何生理紧迫感
- 倒计时人物的台词必须**呼吸短、信息密度低、跳过铺垫直接动作**

**判定**：场景内角色的 stake clock（剩多少命/多少秒/多少米）在台词结构上有体现吗？没有 = Urgency Mismatch。

## Gemini 4 味药方（Polish Agent 重写 SOP）

### 药方 1：名词+动词戒形容词

把比喻句重写为名词+动词驱动的事实句。

| 病句 | 药方 |
|------|------|
| "三年，我擦他们地板上的血" | "我擦地板。我数名字。" |
| "我留着底" | "我有副本。三家事务所。" |
| "你不进的屋子" | "你的会议室。你不去的那间。" |

**操作**：每个形容词/比喻名词（血/屋子/裂缝/底色）问"能换成动词或具体名词吗？" 能 → 换。

### 药方 2：职业化冒犯（古典学身份解构对方黑帮身份）

Sarah 的 Penn Classics 不是装饰，是武器。她的破局方式不是"用学术词表演聪明"，是"用专业**定义**把对方贴的标签重命名"。

**模板**：
- 对方说"我是 Don" → Sarah 说"你不是 Don。podestà——一个被授权的人。Don 自己授权。"
- 对方说"这是命令" → Sarah 说"命令的词根是 mandare——交付。你交付给我的不是命令，是债。"
- 对方说"你是清道夫" → Sarah 说"清道夫的拉丁词根是 emundator——清洁者。你雇我清你不愿见的东西。今晚我把它放回来。"

**为什么有效**：
- 戏内对手听了会愣 0.5 秒（被自己的 frame 解构 = 失重）→ tension 真升级，不是台词显聪明
- 古典学知识在场处境破局 = §15.1 hard rule 真落地，不是 §15.5 装饰
- 0 比喻 / 0 排比 / 0 As-You-Know-Bob——纯定义

### 药方 3：信息差剥夺（show don't tell）

Sarah 的信息优势不通过**说出信息**显示，通过**保留信息**显示。

| 病：报菜名 | 药：保留 |
|---------|--------|
| "副本。我擦过的每一间屋子。每一个名字。每一夜。" | 把 USB 放桌上。"打开它。"（坐下）。让在场所有人盯着 USB。 |
| "一张照片。一辆车。三年前。" | 把照片放在两人之间皮革上。沉默 5 秒。Julian 拿起来。 |

**操作**：每段 Sarah 信息倾倒重写 = 删掉 70% 文字，把信息变成**物**（USB / 照片 / 文件 / 名片）+ Julian 自己读出来的反应。

### 药方 5：让身体接管语言（Body-Mute 解药）

**触发场景**：剧痛 / 戒断 / 流产 / 溺水 / 极度震惊 / 死亡宣判 / 见到尸体 / 被掐脖子 / 性侵后 / 车祸刚下来 等任何**生理失控**时刻。

**操作清单**（按需组合，不是全用）：
- **重复**：单词或短句重复 2-4 次（"No. No. No no no."）
- **未完成**：句子在动词或主语处断掉（"I can't—" / "It's not—you don't—")
- **气音独词**：单词独立成段，前后空行（场景描述里写"她说不出整句"）
- **称呼退化**：成年角色退回到 mom / dad / 全名互称
- **动作覆盖命题**：不写"我害怕"，写"她数地上的瓷砖"；不写"我快不行了"，写"她坐下"
- **卡住句首**：句子起头反复("我——我没——我不知道")
- **声音质地写在台词里**（用括弧标）：(气音) / (吞回去) / (没说完)

**与药方 4 的协作**：
- 药方 4 处理"非身体危机的紧迫场"（递证据 / 对峙 / 倒计时）→ 短句 + 直奔动作 + 单句 ≤15 词
- 药方 5 处理"身体危机的失控场"（剧痛 / 戒断 / 死亡宣判）→ **允许结构破碎**，单句 ≤15 词的硬规则**在这里失效**——句子可以更短（独词）也可以更乱（残句叠残句）

**Sarah 专属说明**：Sarah 是 Penn Classics 学生 + 21 岁 + 6 个月命。她的 voice contract 的"定义精准 + 职业化冒犯"只在**她仍能控制自己的脑子**时生效。当她 (a) 第一次听到诊断 (b) 第一次见 Don 死 (c) Julian 第一次 stake-claim 触碰她 (d) Paolo 假面落地那一刻——这些场景**voice contract 临时挂起**，身体接管语言。Audit 4 测到这些场景如果 Sarah 仍在说精准定义句 = Body-Mute 命中。

### 药方 4：演讲拆成呼吸 ≤15 词

Sarah 单句上限 15 词。超过 15 词 = 拆成"她说一半 → 对方动了一下 → 她说另一半"。**让对方先说，Sarah 只负责掐死下一句。**

**模板对峙**：
```
Julian: "Don't move."（3 词）
Sarah: "你这是 podestà 的口气。"（≤10 词，定义）
Julian: ...（静默，他没听过这词）
Sarah: "Don 不下令。Don 让别人下令。"（≤10 词，再定义一次，把 Julian 重命名）
Julian: "...Calder 小姐。"（5 词，他刚被剥光了）
```

**为什么有效**：
- 句句 ≤15 词 = 0 Trailer-Speak
- 让 Julian 先说 = Sarah 不给观众塞背景，她只回应戏内
- 每句都是**对 Julian** 说，不是对厅里的人/观众说 = 0 As-You-Know-Bob
- Sarah 不做小演讲 = 6 个月命的人 urgency 匹配

## Audit 4 戏剧真实性 rubric（独立 Audit Agent 专用）

Audit 4 Agent 不读 Bible / 骨架 / 其他 audit 报告。**只读最终剧本 + 本文件**。任务：找 Gemini 4 症状每集命中数。

### 强制 grep 清单

每集 ep001/002/003.md 必跑：

```
症状 1 Trailer-Speak：
  grep -E "(三年|今晚|六十秒|三天|十年).{0,20}(三年|今晚|六十秒|三天|十年).{0,20}(三年|今晚|六十秒|三天|十年)" → 三段平行
  grep -E "——.{1,10}秒——" / "——.{1,10}年——" → 倒计时排比
  grep -E "^\s*[一二三四五六七八九十].{1,10}。\s*[一二三四五六七八九十]" → 数词起头排比
  grep -E "^[A-Za-z\u4e00-\u9fa5]{1,8}。\s*[A-Za-z\u4e00-\u9fa5]{1,8}。\s*[A-Za-z\u4e00-\u9fa5]{1,8}。" → 单字句号堆叠

症状 2 Metaphor Overdose：
  per-scene grep "屋子|裂缝|底色|墙|血|裂痕|阴影" 比喻名词数 ≤1

症状 4 Urgency Mismatch：
  Sarah 每场对白平均句长 → 在生死场景（递证据/对峙/倒计时）必须 ≤12 词
```

### 判定标准

| 指标 | PASS | NEEDS_REWORK |
|------|------|-------------|
| Trailer-Speak grep 命中数 | 0 | ≥1 |
| 单场景比喻数 | ≤1 | ≥2 |
| As-You-Know-Bob 人工抽检（每集 ≥3 句长台词逐句过判定测试） | 0 命中 | ≥1 命中 |
| 生死场景 Sarah 平均句长 | ≤12 词 | >12 词 |
| Sarah 古典学 = 破局应用（命名/解构）vs 装饰 | ≥2 处破局 / 0 处装饰 | <2 处破局 或 ≥1 处装饰 |
| **Body-Mute 检测**（生理失控场景 Sarah 是否仍说结构完整句） | 0 命中：失控场出现重复/未完成/气音/动作覆盖命题 | ≥1 命中：失控场全是完整定义句 |

任一指标 NEEDS_REWORK → 整集回 Polish Agent 重写。

### 反 Audit 2 漏洞约束（强制）

Audit 4 **不得**采用 Audit 2 的"像爆款 vs 像 AI slop"盲测——ReelShort/Dark Web 本身就是 Trailer-Speak 代表，"像爆款"不是合格证。Audit 4 只问"戏内人物对戏内对手在这一刻会这么说吗"。

### 反"去情境化反问测试"漏洞约束（强制）

Trailer-Speak 的特征是**情境内也不该说**——"去情境化还能用吗"反问测试抓不到（Trailer-Speak 在剧本和短视频文案里都"能用"，因为它本就是文案）。Audit 4 改用**反向反问**：

> 把这句台词从剧本里抽出来，单独念给一个普通英语观众听——他会觉得这是 (a) 一个真实角色在说一件具体的事 / (b) 一段宣传文案。
>
> (b) = Trailer-Speak。
