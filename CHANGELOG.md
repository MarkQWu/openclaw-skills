# 更新日志

## 2026-03-31 · v1.5

**分镜质量强化 + 自动更新检测**

- 新增**自动更新检测**：使用任何命令时自动检查新版本并提示，不用再手动去 GitHub 看
- 新增 **`/更新` 命令**：一键检查并升级到最新版（支持立即升级、自动更新、暂缓、永不提醒 4 种选择）
- 更新检测支持**国内镜像自动降级**（GitHub 被墙时切 ghfast.top），网络持续 7 天不通会提醒
- `/分镜` 新增**硬约束：台词/音效列保留原文**——禁止缩写、概括或改写剧本台词，台词过长换行显示而非删减
- `/分镜` 新增**景别分布自检**（生成后自动执行）：同景别连续 >3 个标注告警、对话 >8 镜无空镜提示插入、场景开头缺建立镜头提示补充、整集景别占比统计（特写+近景 50-65% / 中景 25-35% / 全景+远景 5-15%）
- `storyboard-guide.md` 新增**景别分布参考数据**（CineScale 数据集 + 竖屏短剧美学研究）和**景别跳转规则**（相邻/跨级/30 度规则）
- VERSION 文件改为纯 semver 格式（`1.5.0`），支持机器解析

## 2026-03-31 · v1.4

**分镜系统全面升级 + 安装体验优化**

- `/分镜` Prompt 体系从「文生图+图生视频」改为**即梦全能多参**（Seedance 2.0），一条 prompt 直接生成视频
- `/分镜` 新增**动态镜头节奏**：按场景情绪自动调节密度（高冲突 30-45 镜/分钟，铺垫 12-20 镜/分钟），不再固定密度
- `/分镜` 新增**台词时长校验**：按每秒 4 字估算，长台词自动拆分多镜头
- `/分镜` 修复蒙太奇段落表格渲染问题（多镜头被压成一行）
- `/分镜` 首次使用时确认镜头节奏偏好，结束后提示可指定总镜头数重拆
- `/帮助` 新增版本号显示
- 安装脚本新增**国内镜像自动降级**（GitHub 连不上时切 ghfast.top）
- 安装脚本新增 WorkBuddy 检测、版本显示、zip 手动下载备选
- `storyboard-guide.md` 全面重写：全能多参公式/示例、多角色处理、关键帧回退条件、常见错误清单

## 2026-03-29 · v1.3

**质量硬约束 + AI Slop 检测 + 自检修复**

- `/分集` 新增 3 条硬约束：单条台词 ≤2 句、每集爽点 ≥3、⚡ 秒级节奏锚点（0-3s 冲突/30s 情绪爆破/结尾钩子）
- `/自检` 新增 **AI Slop 检测**维度：台词书面化扫描、情绪转变过平滑检测、巧合词频统计。总分 60→70
- `/自检` 严重问题自动**定位到场次+台词行**，提供 A/B 修复方案；新增 `--fix` 模式一键修复
- `/导出` 新增**质量门控**：自检 <30 分的集数阻断导出
- `/角色开发` 新增**视觉提示词**字段（10-20 词），作为 `/角色卡` 的种子
- `/角色开发` `/分集` `/自检` 新增**显式视角切换**（人物设计师/职业编剧/质检主管），避免 AI 对自己产出手软

## 2026-03-24 · v1.1

**真实感检查体系**（基于甲方实战反馈提炼）
