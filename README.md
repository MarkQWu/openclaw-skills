# openclaw-skills

gobuildit 社区精选 Skills，兼容 Claude Code 和 OpenClaw。

## 一键安装

前置条件：已安装 [git](https://git-scm.com)。

**Mac / Linux：**

```bash
curl -fsSL https://raw.githubusercontent.com/MarkQWu/openclaw-skills/main/install.sh | bash
```

**Windows（请使用 PowerShell，不支持 cmd）：**

```powershell
irm https://raw.githubusercontent.com/MarkQWu/openclaw-skills/main/install.ps1 | iex
```

安装后关闭当前会话，重新打开 Claude Code / OpenClaw，输入 `/开始` 即可使用。

> 已安装过？再跑一次同样的命令会自动更新到最新版。

## 更新日志

### 2026-03-29 · 爆款剧本工坊 v1.3

**新增：质量硬约束 + AI Slop 检测 + 自检修复**

- `/分集` 新增 3 条硬约束：单条台词 ≤2 句、每集爽点 ≥3、⚡ 秒级节奏锚点（0-3s 冲突/30s 情绪爆破/结尾钩子）
- `/自检` 新增 **AI Slop 检测**维度：台词书面化扫描、情绪转变过平滑检测、巧合词频统计。总分 60→70
- `/自检` 严重问题自动**定位到场次+台词行**，提供 A/B 修复方案；新增 `--fix` 模式一键修复
- `/导出` 新增**质量门控**：自检 <30 分的集数阻断导出
- `/角色开发` 新增**视觉提示词**字段（10-20 词），作为 `/角色卡` 的种子
- `/角色开发` `/分集` `/自检` 新增**显式视角切换**（人物设计师/职业编剧/质检主管），避免 AI 对自己产出手软

### 2026-03-29 · 爆款剧本工坊 v1.2

**新增：剧本→视频全链路（胶水层 + 本体强化）**

- 新增 `/角色卡`：管理角色视觉描述，支持生成/导入两种模式
- 新增 `/分镜 {N}`：剧本→逐镜分镜表 + 即梦 AI prompt，独立可用
- 新增 `/工作流`：打印完整创作→视频链路说明
- 新增 `references/storyboard-guide.md`：分镜拆解指南 + 即梦 prompt 工程 + 20 组情绪→物理动作对照表
- 新增 `scripts/merge_storyboard.py`、`scripts/character_card_validator.py`
- `/分集` 新增**反抽象硬约束**：场景描写禁止纯情绪词，必须转为可拍摄的物理动作
- `/角色开发` 新增**角色-语言风格映射表**（三层映射：角色类型→语言风格→示例台词）
- `/分集` 新增**上下文窗口管理策略**：50+ 集长剧分批加载，保证连贯性

### 2026-03-24 · 爆款剧本工坊 v1.1

**新增：真实感检查体系**（基于甲方实战反馈提炼）

- `/开始` 新增**故事捕获**：先问用户有没有故事想法，有则以此为骨架，不被题材模板覆盖
- `/创作方案` 新增**主题意图确认**、**社交场景预设**、**双层结构**（表面走向 vs 真实走向，适合反转型剧本）
- `/角色开发` 新增**称呼关系表**（N×N 矩阵，防止称呼混乱）、**角色盲点/弱点**（避免脸谱化）、**表面功能 vs 真实功能**
- `/分集` 质量要求新增 5 条真实感检查（称呼逻辑、因果触发、关系生活化、转折铺垫、场景逻辑）
- `/自检` 台词+连贯性维度嵌入真实感子检查项
- 新增 `references/realism-checklist.md` 参考文档，含多题材适用性矩阵
